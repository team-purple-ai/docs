# Welcome to p0

**Build better software, faster.** p0 is Purple's AI-powered desktop coding assistant that takes you from idea to pull request with minimal friction.

p0 uses Claude to understand your codebase, plan changes, and execute implementations across your entire workspace. Whether you're building new features, fixing bugs, or refactoring code, p0 works alongside you with full visibility into every action it takes.
