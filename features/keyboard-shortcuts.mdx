## Essential shortcuts

Here are the most frequently used shortcuts in p0:

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New session | `Cmd+N` | `Ctrl+N` |
| Settings | `Cmd+,` | `Ctrl+,` |
| Toggle sidebar | `Cmd+\` | `Ctrl+\` |
| Command palette | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| Clear chat | `Cmd+K` | `Ctrl+K` |

These shortcuts give you quick access to p0's core functionality:

- **New session** — Start a fresh conversation with Claude
- **Settings** — Open workspace and app settings
- **Toggle sidebar** — Show or hide the conversation history sidebar
- **Command palette** — Access all commands and actions with fuzzy search
- **Clear chat** — Start a new sub-chat in the current session (preserves context)

## Chat shortcuts

When working in the chat interface:

- **Enter** — Send message (when chat input is focused)
- **Shift+Enter** — Add new line without sending
- **Cmd+K** (macOS) / **Ctrl+K** (Windows/Linux) — Clear chat and start fresh

## Terminal shortcuts

The embedded terminal supports standard terminal shortcuts:

- **Cmd+T** (macOS) / **Ctrl+Shift+T** (Windows/Linux) — New terminal tab
- **Cmd+W** (macOS) / **Ctrl+W** (Windows/Linux) — Close current terminal tab
- **Cmd+F** (macOS) / **Ctrl+Shift+F** (Windows/Linux) — Search in terminal

## Customization

To view the complete list of shortcuts or customize them:

1. Open the command palette with `Cmd+Shift+P` (macOS) or `Ctrl+Shift+P` (Windows/Linux)
2. Type "shortcuts" to find the keyboard shortcuts dialog
3. Browse all available shortcuts organized by category

For the complete reference of all shortcuts in p0, see [all shortcuts](/reference/all-shortcuts).
