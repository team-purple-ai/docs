# Installation

Get p0 running on your development machine in minutes. The desktop app is available for all major platforms with native builds optimized for each operating system.

## Download and install

<Tabs>
  <Tab title="macOS">
    ### Download for macOS

    p0 supports both Apple Silicon (M1/M2/M3/M4) and Intel-based Macs running macOS 12.0 (Monterey) or later.

    1. Download the `.dmg` installer from [bepurple.ai](https://bepurple.ai)
    2. Open the downloaded `.dmg` file
    3. Drag the p0 icon into your Applications folder
    4. Launch p0 from Applications or Spotlight

    <Note>
      On first launch, macOS may show a security prompt since p0 is downloaded from the web. Click **Open** in the dialog, or go to **System Settings > Privacy & Security** and click **Open Anyway**.
    </Note>
  </Tab>

  <Tab title="Windows">
    ### Download for Windows

    p0 supports Windows 10 and Windows 11 (64-bit).

    1. Download the NSIS installer from [bepurple.ai](https://bepurple.ai)
    2. Run the `p0-setup.exe` file
    3. Follow the installation wizard
    4. Launch p0 from the Start menu or desktop shortcut

    **Administrator rights**: The installer may request administrator privileges to install system-level components.
  </Tab>

  <Tab title="Linux">
    ### Download for Linux

    p0 is available as an AppImage or .deb package for Ubuntu, Debian, and other distributions.

    **AppImage** (universal):
    ```bash
    chmod +x p0.AppImage
    ./p0.AppImage
    ```

    **Debian/Ubuntu** (.deb package):
    ```bash
    sudo dpkg -i p0.deb
    sudo apt-get install -f  # Install dependencies if needed
    ```

    Launch p0 from your application menu or run `p0` from the terminal.
  </Tab>
</Tabs>

## First launch

When you open p0 for the first time, you'll be prompted to sign in with your Purple account.

  <img src="/images/screenshots/login-screen.png" alt="p0 login screen" />

<Steps>
  <Step title="Sign in or create an account">
    Enter your email and password to sign in. If you're new to Purple, switch to the **Create Account** tab to register with your email, password, and name.
  </Step>

  <Step title="Connect to Claude">
    p0 securely authenticates with Anthropic's Claude API. Your credentials are encrypted using your operating system's keychain (macOS Keychain, Windows DPAPI, or Linux Secret Service).
  </Step>

  <Step title="You're in">
    After signing in, you'll land on the workspace picker where you can create your first workspace or clone one from your team.
  </Step>
</Steps>

<Note>
  Don't have a Purple account yet? Sign up at [bepurple.ai](https://bepurple.ai) to get started.
</Note>

## Auto-updates

p0 checks for updates automatically in the background. When a new version is available, you'll see a notification with the release notes. Updates are downloaded in the background and installed the next time you quit and reopen the app.

You don't need to do anything to stay up to date â€” just restart p0 when prompted.
