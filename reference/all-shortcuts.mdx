All keyboard shortcuts available in p0. macOS uses `Cmd`, Windows and Linux use `Ctrl`.

## General

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New session | `Cmd+N` | `Ctrl+N` |
| Settings | `Cmd+,` | `Ctrl+,` |
| Command palette | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| Toggle sidebar | `Cmd+\` | `Ctrl+\` |
| Focus chat input | `Cmd+L` | `Ctrl+L` |
| Send message | `Cmd+Enter` | `Ctrl+Enter` |
| Close dialog / Cancel | `Escape` | `Escape` |

## Tabs

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New tab | `Cmd+T` | `Ctrl+T` |
| Close tab | `Cmd+W` | `Ctrl+W` |
| Switch to tab 1-9 | `Cmd+1` through `Cmd+9` | `Ctrl+1` through `Ctrl+9` |
| Previous tab | `Cmd+Shift+[` | `Ctrl+Shift+[` |
| Next tab | `Cmd+Shift+]` | `Ctrl+Shift+]` |
| Quick switch | `Ctrl+Tab` | `Ctrl+Tab` |

## Chat

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Clear / new sub-chat | `Cmd+K` | `Ctrl+K` |
| Compact context | `Cmd+Shift+K` | `Ctrl+Shift+K` |

<Tip>
You can customize the `Ctrl+Tab` behavior in Settings > Preferences to switch between workspaces or agents.
</Tip>
